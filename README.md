# WWN Character Creator
In-browser app to create characters in the *Sine Nominee* tabletop roleplaying game ***Worlds Without Number***. All information taken from the publicly-available version provided by Kevin Crawford available [here](https://drive.google.com/file/d/14qSGink8i1Y4ULaqdivrcvyKNI6Rkbes/view). This code is copied verbatim from Noah Sennett, account "noah-sennett", GPL 3 license.

**Use the tool by clicking [here](https://krisschnee.github.io/wwn-character-creator/).**
